# React Simple

A simple React application built with Vite that demonstrates core React concepts including state management with hooks, user input handling, and dynamic rendering.

## Features

- **Counter** - Increment, decrement, and reset a counter value
- **Greeting** - Dynamic greeting that responds to user input
- **Todo List** - Add and remove items from a todo list

## Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- npm (comes with Node.js)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/hemantbobade/react-simple.git
   cd react-simple
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

## Usage

### Development

Start the development server with hot module replacement:

```bash
npm run dev
```

The application will open automatically in your browser at `http://localhost:5173`.

### Production Build

Build the application for production:

```bash
npm run build
```

### Preview Production Build

Preview the production build locally:

```bash
npm run preview
```
